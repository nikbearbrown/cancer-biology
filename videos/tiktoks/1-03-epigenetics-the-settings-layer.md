## TikTok 1.3 — Epigenetics: The Settings Layer

**Book:** Cancer Biology  
**Chapters:** 7-8  
**Length target:** 65 seconds  
**Learning promise:** viewers understand epigenetics as altered gene reading rather than altered DNA letters.

**Hook:** "Cancer can happen when the DNA text is intact but the settings are wrong."

**Shot-by-shot timeline:**

| Time | Visual | Voiceover | On-screen text |
|---:|---|---|---|
| 0-4 | Open book labeled "DNA." Text is readable. | "Mutations change DNA letters." | Mutation = letter change |
| 4-10 | Sliders appear: off/on, quiet/loud. | "Epigenetics changes how the cell reads those letters." | Epigenetics = reading settings |
| 10-22 | Methyl tags silence a tumor suppressor gene. | "A protective gene can be physically present but methylated into silence." | Present but silenced |
| 22-34 | Histone coil tightens and loosens. | "Chromatin can close around genes or open them for transcription." | Chromatin controls access |
| 34-46 | Noncoding RNA icons tune several genes. | "Noncoding RNAs can tune whole networks without becoming proteins." | RNA can regulate |
| 46-60 | Three wrong settings appear in a cancer cell. | "So cancer is not only misspelled genes. It is also wrong settings: protective genes off, growth programs on, memory systems distorted." | Wrong settings can be malignant |
| 60-65 | End card. | "That is the control layer above the code." | Cancer Biology, Ch. 7-8 |

**Full voiceover:**
"Mutations change DNA letters. Epigenetics changes how the cell reads those letters. That matters in cancer because a protective gene can be physically present but methylated into silence. Chromatin can close around genes or open them for transcription. Noncoding RNAs can tune whole networks without becoming proteins. So cancer is not only misspelled genes. It is also wrong settings: protective genes turned off, growth programs turned on, and cellular memory systems pushed into a malignant state."

**Asset prompts:**
- "DNA as a book with control sliders, educational biomedical illustration."
- "DNA methylation tags silencing a tumor suppressor gene, clean flat medical graphic."

**Caption copy:** Epigenetics is the control layer above the code. In cancer, the settings can become part of the disease.
